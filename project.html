Project 01

WineCellar

JavaScript

React.js

Node.js

MongoDB

A virtual wine cellar application that allows user to store wine information & photos in their private online cellar, as well as search the web for best prices of chosen wine using custom wine finder tool.

Main functionalities of the application:

"Creating user profile,", "Creating personal cellar - a collection of wines,", "Adding newly discovered wines to cellar - with photos and detailed information,", "Viewing details of chosen wine,", "Using the wine finder tools allowing for searching & comparing chosen wine prices in online wine stores."
